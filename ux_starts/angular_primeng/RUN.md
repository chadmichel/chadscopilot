# Running the Angular PrimeNG Starter

This project uses Angular and PrimeNG with MSW (Mock Service Worker) for API mocking.

## Prerequisites
- Node.js (v18 or later)
- npm

## How to Run

### 1. Install Dependencies
```bash
npm install
```

### 2. Run with Mock Configuration (Recommended)
To run the application with mocked API responses:
```bash
npm run start:mock
```
The application will be available at `http://localhost:7777`.

### 3. Run Standard Development Server
To run without the mock configuration:
```bash
npm start
```
The application will be available at `http://localhost:7020`.
